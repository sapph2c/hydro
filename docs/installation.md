# Installation

Hydro is available as a Python package on PyPI and can be installed using `uv` (recommended), `pipx`, or `pip`.

**Note**: python 3.13+ is required

Install using uv:

```bash
uv install tool hydro@latest
```

Install using pipx:

```bash
pipx install hydro
```

Install using pip:

```bash
pip install hydro
```
